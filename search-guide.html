语料搜索系统

Search Interface — How to Use

无需下载数据，无需编程，直接在网页上检索全部语料。

系统简介

语料搜索系统是一个静态网页，让你在浏览器里直接搜索数据集中的全部文本内容，无需打开任何数据文件。

主要功能：

  • 输入关键词，即时搜索全部 311 个文件的内容
  • 按数据类别筛选（两会/白皮书/领导人著作等）
  • 按发布时间范围筛选
  • 搜索结果显示文件名、日期和相关文本片段
关于数据保护

搜索系统只展示每条记录约 200 字的文本片段，不提供完整原文下载。这样既能帮助研究者定位相关资料，也保护了完整数据集的完整性。

在线版（推荐）

访问在线版无需任何安装：

→ 打开搜索系统

本地运行

如果在线版暂时不可用，可以在本地运行：

第一步：安装 Python 依赖

pip install pandas openpyxl

第二步：生成搜索索引

在终端中，进入数据集根目录，运行：

python search/build_index.py

这个脚本会读取全部数据文件，生成 search/search_index.json。根据电脑性能，大约需要 1-3 分钟。

第三步：打开搜索界面

用任意浏览器打开 search/index.html 文件即可。

提示

双击 search/index.html 有时会因为浏览器安全限制无法加载本地 JSON 文件。如果遇到这种情况，可以用以下方式：

# 在 search/ 目录下启动一个简单的本地服务器
cd search
python -m http.server 8080
# 然后在浏览器访问 http://localhost:8080

如何使用检索界面

基础搜索

在顶部搜索框输入关键词，结果会即时更新。

搜索支持： - 单个词：共同富裕 - 多个词（空格隔开，结果需同时包含）：腐败 治理 - 短语：依法治国

按类别筛选

点击右侧下拉菜单，选择只在某类文件中搜索：

选项 对应数据
习近平讲话 习近平系列重要讲话数据库
人民网政绩观 人民网政绩观专题文章
领导人著作 各领导人著作选集
党代会 全国党代会文件
两会 全国两会文件
白皮书 政府白皮书
党规党史 党规党史与五年规划
意识形态 意识形态文件
求是 《求是》杂志

按时间筛选

在「年份范围」输入框中填写起止年份，如 2012 至 2020。

搜索结果说明

每条搜索结果包含：

两会 政府工作报告 2021
2021政府工作报告.xlsx
…扎实推动共同富裕，完善分配制度，增加低收入群体收入，扩大中等收入群体，推进基本公共服务均等化…
  • 类别徽章：颜色标识数据类别
  • 标题：文件名称或文章标题
  • 日期：发布年份（部分数据有精确日期）
  • 来源文件：原始文件名
  • 文本片段：包含关键词的上下文（关键词高亮显示）

使用场景示例

场景一：找到「共同富裕」最早出现在什么文件里

搜索「共同富裕」，将年份筛选设为 1998 至 2015，查看哪些早期文件中有这个词，以及当时的语境是什么。

场景二：查找反腐相关段落，用于对话分析

搜索「反腐」，类别选「习近平讲话」，找到相关片段后，把文本粘贴给配合技能的大模型进行深度分析。

场景三：对比同一词在不同类别文件中的使用

先搜索某词，类别选「两会」，记录结果；再改选「白皮书」，对比同一概念在不同类型官方文件中的叙述差异。
