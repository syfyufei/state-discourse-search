数据集概览

Dataset Overview — What’s Inside

本章介绍数据集中九大类别的具体内容、字段说明和适用研究方向。

总体规模

指标 数值
总文件数 311 个（308 个 Excel + 2 个 CSV）
总体量 约 47.5 MB
时间跨度 1950 年代 — 2026 年
语言 中文

类别一：习近平系列重要讲话数据库

文件：习近平系列重要讲话数据库.csv 规模：48,610 条记录

这是数据集中规模最大、结构最规整的数据，以 CSV 格式存储，每行对应一篇讲话或文章。

字段名 说明
标题 讲话或文章的标题
中文 正文全文
发布时间 发布日期
字段 主题领域分类（如：经济、外交、党建）
标题链接 原文网址

适合研究：习近平执政话语特征、特定议题（经济/外交/反腐/生态）关键词频次、话语风格演变

类别二：人民网政绩观专题文章

文件：人民网_政绩观.csv 规模：194 条记录（2025-2026 年）

《人民日报》专题报道，聚焦「正确政绩观」这一近期核心话语。

字段名 说明
title 文章标题
full_text 正文全文
date 发布日期
source 来源（人民日报等）
snippet 摘要
url 原文链接

适合研究：当代党政绩效评价话语、反腐廉政叙事、地方治理话语

类别三：领导人著作选集

文件：32 个 Excel 文件，共 25+ 卷

跨越七十余年的领导人著作，是做跨时期话语比较的核心数据来源。

领导人 文本 大致时代
毛泽东 毛泽东选集（第 1-5 卷） 1920s-1970s
刘少奇 刘少奇选集（上下册） 1940s-1960s
周恩来 周恩来选集（上下册） 1940s-1970s
朱德 朱德选集 1920s-1970s
陈云 陈云文选（第 1-3 卷） 1950s-1980s
邓小平 邓小平文选（第 1-3 卷） 1950s-1990s
江泽民 江泽民文选（第 1-3 卷） 1989-2002
习近平 习近平谈治国理政（第 1-3 卷） 2012-
习近平 习近平论述摘编（深化改革 / 中国梦 / 新冠与经济 / 法治 / 反腐 / 人权） 2012-

通用字段：中文（正文全文）、书名（来源卷册）

适合研究：跨领导人话语比较、意识形态演变谱系、特定概念（改革/法治/民主/人权）的话语建构

类别四：全国党代会文件

文件：39 个 Excel 文件

会议 时间 主要文件类型
十六大 2002 政治报告、纪检报告、党章修正案
十七大 2007 政治报告、纪检报告、党章修正案
十八大 2012 政治报告
十九大 2017 政治报告、纪检报告、党章修正案
十八届各全会 2014-2016 公报、决定、准则
十九届各全会 2020-2021 公报、决议

通用字段：中文（全文）、文件名称（文件类型）

适合研究：路线方针五年演变、党建话语制度化、重大政治转向的话语分析

类别五：全国两会文件

文件：130 个 Excel 文件（1998-2023，连续 26 年）

这是整个数据集中时间序列最完整的部分，每年五份核心文件：

  • 政府工作报告
  • 国民经济和社会发展计划报告
  • 中央财政预算报告
  • 全国人大常委会工作报告
  • 全国政协工作报告

通用字段：中文（全文）、文件名称（文件类型）、发表时间（年份）

适合研究：政策话语年度演变、经济目标表述变化、话语风格历时分析

类别六：政府白皮书

文件：89 个 Excel 文件（2010-2021）

国务院新闻办公室发布的权威官方政策阐释文件，议题广泛：

反腐廉政、对外贸易、人力资源、法律体系、农村扶贫、 国防（2010/2012/2014）、气候变化、航天、西藏、 医疗卫生、司法改革、钓鱼岛主权、稀土、能源等

适合研究：官方叙事框架分析、敏感议题的话语建构、对外宣传话语策略

类别七：党规党史与五年规划

文件：14 个 Excel 文件

党规（7 份）：党章、党内政治生活准则、廉洁自律准则、纪检工作规则、问责条例、党内监督条例、纪律处分条例

党史（5 份）：中国共产党历史（第 1-2 卷）、党史七十年、中国共产党简史、抗日战争史简明读本

五年规划（2 份）：十三五（2016-2020）、十四五（2021-2025 附 2035 远景目标）

适合研究：制度性话语分析、纪律话语演变、党的历史叙事建构、中长期目标话语

类别八：意识形态与发展理念文件

文件：3 个 Excel 文件

  • 科学发展观关键语句（胡锦涛时代核心话语精炼版）
  • 中共中央关于完整准确全面贯彻新发展理念做好碳达峰碳中和工作的意见
  • 2030 年前碳达峰行动方案

适合研究：发展理念话语演变、生态文明话语的意识形态建构、政策目标的话语呈现

类别九：《求是》杂志

文件：42 个 Excel 文件（多期）

《求是》是中共中央主办的旗舰理论刊物，专门负责对党的路线方针进行理论阐释和意识形态宣传。

通用字段：中文（正文）、文件名称（期号）

适合研究：官方理论话语体系、意识形态宣传机制、政治话语的学术化过程

如何选择适合自己研究问题的数据？

研究问题类型 推荐数据类别
追踪某个词/概念随时间的变化 两会文件（1998-2023）、习近平讲话数据库
比较不同领导人的话语风格 领导人著作选集
分析党的路线方针演变 党代会文件、党规党史
研究某个政策议题的官方叙事 政府白皮书、习近平讲话数据库
研究意识形态与理论话语 《求是》杂志、意识形态文件、领导人著作
当代治理与反腐话语 人民网政绩观、习近平讲话数据库、党规
