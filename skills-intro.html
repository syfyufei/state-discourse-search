技能提示词：是什么，怎么用

Skill Prompts — Introduction

Skills are pre-written prompts that turn any LLM into a specialized political discourse research assistant.

技能提示词是什么？

「技能提示词」（Skill Prompt）是一段预先写好的指令文字。当你把它粘贴给大模型时，大模型就会变成一位了解这份数据集全部内容、能主动引导你做研究的助手。

你不需要向大模型解释数据集有什么——这些信息已经写在技能里了。你只需要告诉它你想研究什么。

为什么需要技能提示词？

直接问大模型「帮我分析习近平讲话」，它不知道你的数据是什么格式、有哪些字段、覆盖什么时间范围。

使用技能提示词后，大模型会：

  1. 主动引导你：询问你的研究问题、时间范围和兴趣方向
  2. 推荐合适的数据：告诉你应该用哪个文件的哪些字段
  3. 选择分析方式：对话分析（直接讨论）或代码生成（输出可运行的Python/R代码）
  4. 输出学术成果：分析结论、原文引用、可用于论文的段落

共有几个技能，分别适合什么？

本数据集提供四个技能，分别针对不同研究场景：

技能文件 适合场景 推荐指数
skill_general.md 通用入口，不知道从哪里开始时先用这个 ⭐⭐⭐⭐⭐ 首选
skill_policy.md 研究某个政策话题随时间的演变 ⭐⭐⭐⭐
skill_ideology.md 比较不同领导人话语，或研究意识形态演变 ⭐⭐⭐⭐
skill_codewriter.md 有Python环境，想自己跑数据生成图表 ⭐⭐⭐

通用使用步骤

所有技能的使用方式完全一样，只是内容不同：

四步上手

第一步：打开技能文件

在 skills/ 目录下找到对应的 .md 文件，全选所有内容并复制。

第二步：打开大模型对话

打开 Claude / ChatGPT / DeepSeek / Gemini 等任意大模型，新建一个对话。

第三步：粘贴技能，发送

把复制的内容粘贴到输入框里，直接发送。大模型会确认它已了解数据集，并询问你的研究问题。

第四步：告诉它你想研究什么

用自己的话描述研究问题，例如：

  • “我想看看’共同富裕’这个词是从什么时候开始出现的”
  • “我要比较毛泽东和习近平谈’人民’的方式有什么不同”
  • “帮我分析历年政府工作报告中关于经济增长目标的表述”

大模型会接着引导你完成分析。

技能支持哪两种工作模式？

每个技能都支持两种模式，在对话中你可以告诉大模型你想用哪种：

模式一：对话分析（无需编程）

你把相关文本片段直接粘贴给大模型，它帮你： - 识别关键词和话语框架 - 进行跨文本比较分析 - 提取核心论点和修辞策略 - 生成可用于论文写作的学术段落

适合：没有编程经验的研究者，或者只需要分析少量文本的情况

模式二：代码生成（需要 Python 环境）

大模型根据你的研究问题，生成完整的 Python 分析代码，你直接复制运行即可得到结果（图表、统计数据等）。

如需调用大模型 API 进行批量文本分析，大模型会提示你输入 API Key（支持 OpenAI / Anthropic Claude 等）。

适合：需要处理大量文本，或需要生成可视化图表的研究者
