State Discourse Dataset — A text corpus for Chinese political discourse research, with LLM-powered skill prompts.

这本手册是什么？

这是「国家话语数据集」（State Discourse Dataset）的完整使用手册。

这份数据集收录了从 1950 年代至 2026 年的中国官方政治话语文本，共约 311 个文件、47.5 MB，涵盖领导人著作、党代会文件、两会报告、政府白皮书等九大类别。

这本手册专门写给没有数据分析基础的研究者。 你不需要会编程，不需要懂统计，只需要一个大模型（如 ChatGPT、Claude、DeepSeek）和一个研究问题，就能开始分析。

你能用这份数据做什么？

示例研究问题
  • 「共同富裕」这个词是从什么时候开始频繁出现在官方话语中的？
  • 毛泽东和习近平在谈论「人民」时，语境和含义有什么不同？
  • 历年政府工作报告中，经济目标的表述方式如何变化？
  • 中国政府在白皮书中如何建构「人权」话语？
  • 反腐话语在十八大前后发生了什么变化？

如何使用这本手册？

这份数据集提供三种使用方式，你可以根据自己的需求选择：

方式一：技能提示词（最推荐）

把「技能提示词」（Skill Prompt）复制粘贴给任何大模型，让大模型引导你完成研究。完全不需要编程。

→ 从 技能提示词指南 开始

方式二：语料搜索系统

在网页上直接检索全部语料，按关键词、类别、时间筛选，浏览相关文本片段。

→ 查看 搜索系统使用说明

方式三：Python 数据加载（有基础编程经验）

如果你有 Python 环境，可以用提供的脚本加载和筛选数据，进行自定义分析。

→ 查看 数据加载脚本说明

五分钟快速上手

  1. 先读概览：浏览 数据集概览，看看哪类数据和你的研究方向相关
  2. 选一个技能：去 技能提示词指南 选一个合适的技能文件
  3. 打开大模型：Claude、ChatGPT、DeepSeek 等都可以
  4. 粘贴技能，开始对话：大模型会引导你一步步完成分析

就这么简单。
