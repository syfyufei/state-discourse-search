数据加载脚本

Data Loading Script — For Users with Python

本章适合有基础 Python 环境的用户。如果你没有编程经验，跳过本章，直接使用 技能提示词 即可。

为什么需要加载脚本？

数据集包含 308 个分散的 Excel 文件。如果想批量分析多个文件（比如「加载全部 26 年的两会数据」），手动打开每个文件会非常繁琐。

scripts/load_data.py 提供了简单的函数，让你用一行代码加载整类数据。

安装依赖

pip install pandas openpyxl

基础用法

查看所有可用类别

import sys
sys.path.insert(0, '.')  # 在数据集根目录运行
from scripts.load_data import list_categories

list_categories()

输出：

加载某类数据

from scripts.load_data import load_category

# 加载全部两会数据
df = load_category('两会')
# 输出：已加载 '两会'：XXXXX 条记录（来自 130 个文件）

# 加载习近平讲话
df = load_category('习近平讲话')

# 加载白皮书
df = load_category('白皮书')

筛选数据

# 按关键词筛选
df = load_category('两会', keyword='共同富裕')

# 按年份范围筛选
df = load_category('两会', year_start=2012, year_end=2020)

# 同时筛选
df = load_category('习近平讲话', keyword='反腐', year_start=2013, year_end=2023)

统计关键词年度频次

from scripts.load_data import load_category, count_keyword_by_year

# 加载数据
df = load_category('习近平讲话')

# 统计年度词频
trend = count_keyword_by_year(df, '共同富裕')
print(trend)

导出为 CSV

from scripts.load_data import load_category, export_to_csv

df = load_category('两会', keyword='改革')
export_to_csv(df, '两会_改革相关段落.csv')
# 输出：已导出：两会_改革相关段落.csv（XXXX 条记录）

导出的 CSV 可以用 Excel 打开，也可以直接复制内容粘贴给大模型进行分析。

与代码生成技能配合使用

如果你使用 代码生成技能（skill_codewriter），大模型生成的代码会自动导入本脚本中的函数，让数据加载部分更简洁：

# 大模型生成的代码通常会这样调用
import sys
sys.path.insert(0, '/path/to/StateDiscourseDataset')
from scripts.load_data import load_category, count_keyword_by_year
